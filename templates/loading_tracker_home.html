How it works

Weekly workflow, simplified

1. Build or refresh template Use a good live week and save it as the active backend planning template.
2. Start week from template Future weeks open from the saved structure instead of relying on a fresh workbook upload.
3. Carry forward what matters Pending rows and remaining stock can still travel into the next week automatically.
4. Import only as fallback Weekly workbook import stays available when the business structure changes sharply.
